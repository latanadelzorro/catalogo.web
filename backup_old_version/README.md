# Chef El Zorro - Versión Profesional

Este es el sitio web oficial de Chef El Zorro, una repostería artesanal de alta calidad. Este repositorio contiene la versión profesional del sitio web.

## Características

- **Diseño profesional y elegante**
- **Totalmente responsivo** (funciona en móviles, tablets y escritorio)
- **Optimizado para SEO**
- **Rápido y ligero**
- **Fácil de mantener y actualizar**

## Estructura del Proyecto

```
profesional/
├── index.html          # Página principal
├── styles.css         # Estilos principales
├── script.js          # Funcionalidades JavaScript
└── README.md          # Este archivo
```

## Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Personaliza el contenido en `index.html` según sea necesario
4. Modifica los estilos en `styles.css`
5. Añade nuevas funcionalidades en `script.js`

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (para cargar fuentes de Google e iconos de Font Awesome)

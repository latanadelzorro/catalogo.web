# Brief para el Sitio Web de El Zorro Repostería

## Descripción General
Sitio web estático para "El Zorro Repostería" que muestra una selección de productos de repostería artesanal. El diseño es limpio, moderno y completamente responsivo.

## Estructura del Sitio
- Página de inicio con sección hero y catálogo de productos
- Diseño de una sola página con scroll suave
- Sin dependencias de JavaScript (solo HTML y CSS)

## Características Principales
1. **Encabezado Fijo**
   - Logo a la izquierda
   - Menú hamburguesa desplegable a la derecha
   - Diseño compacto que se reduce en scroll

2. **Sección Hero**
   - Imagen de fondo con superposición degradada
   - Título y texto descriptivo centrados
   - Botón de llamada a la acción

3. **Catálogo de Productos**
   - Grid responsivo de tarjetas de productos
   - Cada producto incluye:
     - Imagen
     - Categoría
     - Nombre
     - Descripción
     - Ingredientes
     - Alérgenos con iconos

4. **Diseño Responsive**
   - Adaptable a móviles, tablets y escritorio
   - Menú adaptativo
   - Imágenes optimizadas
   - Tipografía legible en todos los dispositivos

## Paleta de Colores
- Principal: #2C1810 (Marrón chocolate oscuro)
- Secundario: #8B5A2B (Marrón caramelo)
- Acento: #D4A76A (Dorado claro)
- Fondo: #FFF9F2 (Blanco roto)
- Texto: #333333 (Gris oscuro)

## Tipografía
- Títulos: 'Playfair Display', serif
- Texto: 'Montserrat', sans-serif

## Estructura de Archivos
```
/
├── index.html          # Página principal
├── styles.css         # Estilos principales
├── imagenes/          # Carpeta de imágenes
│   ├── hero-bg.jpg    # Fondo de la sección hero
│   ├── icono_vectorizado.svg  # Logo
│   └── productos/     # Imágenes de los productos
└── refactorizacion/   # Versión optimizada
    ├── index.html
    └── styles.css
```

## Requisitos Técnicos
- HTML5 semántico
- CSS3 con variables personalizadas
- Flexbox y Grid para el diseño
- Iconos de Font Awesome
- Fuentes de Google Fonts
- Compatibilidad con navegadores modernos

## Instrucciones de Instalación
1. Clonar el repositorio
2. Abrir `index.html` en un navegador web
3. Para desarrollo, usar un servidor local

## Notas Adicionales
- Todas las imágenes deben estar optimizadas para web
- Implementar metaetiquetas para SEO básico
- Asegurar la accesibilidad (etiquetas alt, ARIA, etc.)
- El sitio no requiere backend ni base de datos
